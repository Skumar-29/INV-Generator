Invoice Generator App - v7 Rate Dropdown + Add Items

Upload all files inside this folder to the root of your GitHub repository.
Do not upload only the ZIP file.

Important updates in this version:
- Rate boxes in the Load table now work like the other dropdown-enabled boxes.
- Single tap/click on the Rate box is for manual entry.
- Double tap/double click on the Rate box opens the saved rate dropdown list.
- Load Rate dropdown uses rates from the BD/RT/BT/AB Auto Rates settings.
- Miscellaneous Rate dropdown uses rates from the Miscellaneous Auto Rates settings.
- BD/RT/BT/AB Auto Rates now has an Add category/rate option directly under the rate table.
- Adding a BD/RT/BT/AB category in the Auto Rates area also adds it to the invoice BD/RT/BT/AB dropdown list.
- Miscellaneous Auto Rates now has an Add item/rate option directly under the rate table.
- Adding a miscellaneous item/rate also adds it to the Miscellaneous Item dropdown list.
- Selecting a BD/RT/BT/AB item still auto-fills its rate and recalculates the amount.
- Selecting a Pick-up/Delivery/miscellaneous item still auto-fills its rate and recalculates the amount.
- Existing backup, import, update app, invoice list/grid, PDF, Excel, and email features remain.

After uploading to GitHub:
1. Open the repository.
2. Upload/replace index.html, manifest.json, sw.js, icon.svg, and README.txt in the root.
3. Commit changes.
4. Wait 1-5 minutes for GitHub Pages to update.
5. Open the app and press Settings > Update App, or open the app link with ?v=7 at the end.

Example:
https://your-app-link/?v=7
