Invoice Generator App

How to use:
1. Open index.html in a browser, or upload the whole folder to GitHub Pages.
2. On iPhone Safari, open the hosted link and choose Share > Add to Home Screen.
3. Use the Settings screen to update default top details, next invoice number, drop-down lists, city short names, rates, and miscellaneous items.
4. Create invoices from the Invoice screen. Start/finish odometer calculates KM and amount automatically when a numeric rate is used.
5. Saved invoices appear under All Invoices in List or Grid view.
6. Export PDF or Excel from the invoice form, preview screen, or invoice history screen.

Notes:
- Data is saved locally in the browser using localStorage.
- Exported PDF uses the app's built-in PDF writer and is named as Invoice-number + date range.
- Exported Excel is an Excel-readable .xls file.
